# Fake News Detector
Fake News Detector is a machine learning based web application that analyzes news headlines or statements and predicts whether they are real or fake using Natural Language Processing (NLP) techniques.

## Features
- Detects fake or real news
- Fast prediction results
- Clean user-friendly interface
- Built using Flask
- Uses machine learning model

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## How to Run
1. Install required libraries
pip install flask scikit-learn pandas numpy

2. Run project
python detection.py

3. Open browser
http://127.0.0.1:5000/
